# Трекер прочитанных книг

Консольное приложение на Python для хранения списка прочитанных книг в JSON-файле.

## Возможности

- добавление книги;
- просмотр всех книг;
- расчёт средней оценки;
- статистика по авторам;
- удаление книги;
- защита от добавления дублей по паре автор + название;
- хранение данных в файле `books.json`.

## Поля книги

- автор;
- название;
- оценка от 1 до 5;
- дата прочтения.

## Как запустить

```bash
python main.py
```

## Пример меню

```text
1. Добавить книгу
2. Показать все книги
3. Показать среднюю оценку
4. Статистика по авторам
5. Удалить книгу
6. Выход
```

## Пример работы

```text
Выберите пункт меню: 1
Автор: Рэй Брэдбери
Название: 451 градус по Фаренгейту
Оценка (1-5): 5
Дата прочтения: 2026-05-19
Книга добавлена.
```
